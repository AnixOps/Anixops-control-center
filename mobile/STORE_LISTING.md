### Full Description (4000 characters max)
```
AnixOps Control Center is a powerful, unified platform for managing your entire infrastructure from anywhere. Whether you're a DevOps engineer, system administrator, or IT manager, AnixOps puts you in control.

KEY FEATURES:

📊 Real-time Dashboard
• Monitor server health and performance at a glance
• View node statistics and user activity
• Customizable widgets for your workflow

🖥️ Server Management
• Add, configure, and manage multiple servers
• Start, stop, and restart nodes remotely
• View detailed server logs and metrics

📋 Ansible Automation
• Run Ansible playbooks directly from your mobile device
• Schedule automated tasks
• View playbook execution history

👥 User Management
• Manage user accounts and permissions
• Role-based access control
• Multi-factor authentication support

🔌 Plugin System
• Extend functionality with plugins
• v2board and V2bX panel integration
• AnixOps agent remote control

🔒 Security First
• End-to-end encryption
• Biometric authentication (Face ID/Touch ID)
• Secure credential storage

📱 Multi-platform
• Seamless experience across mobile, web, and desktop
• Offline mode with automatic sync
• Push notifications for alerts and events

🌙 Modern UI
• Dark and light themes
• Material Design 3
• Accessibility support
• Multi-language support (English, Chinese, Japanese, Arabic)

USE CASES:
• Monitor server infrastructure on the go
• Respond to incidents quickly with push notifications
• Execute automation tasks remotely
• Manage multiple servers from a single dashboard
• Collaborate with team members

For more information, visit: https://github.com/AnixOps/anixops-control-center

Support: support@anixops.com
```
