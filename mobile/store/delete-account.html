Request Account Deletion

Last updated: March 17, 2026

⚠️ Warning: Account deletion is permanent and cannot be undone. All your data will be removed from our systems.

How to Request Account Deletion

  1. Email us at privacy@anixops.com
  2. Subject line: "Account Deletion Request - AnixOps Control Center"
  3. Include in your email:
    • Your registered email address
    • Username (if applicable)
    • Reason for deletion (optional)
  4. Processing time: We will process your request within 30 days
  5. Confirmation: You will receive an email confirmation once deletion is complete

Alternative Methods

  • GitHub Issue: Create an issue on our GitHub repository
  • In-App: Go to Settings → Account → Delete Account

What Data Will Be Deleted

Data Type Deleted? Notes
Account Information ✅ Yes Email, username, profile data
Authentication Data ✅ Yes Password hash, tokens
User Preferences ✅ Yes Settings, preferences
Server Configurations ✅ Yes Your saved server connections
Activity Logs ✅ Yes Your usage history
Analytics Data ✅ Yes Anonymized within 30 days

Data Retention

Some data may be retained for legal or security purposes:

  • Security logs: Retained for 90 days for fraud prevention
  • Legal compliance: As required by applicable laws
