# AnixOps Control Center - Closed Testing

## 测试版本信息
- **版本**: 1.0.0-alpha.1 (Build 2)
- **测试类型**: Closed Testing (Alpha)
- **测试期限**: 14天

## 测试说明

### 测试目标
测试AnixOps移动端应用的核心功能，确保在生产环境发布前达到稳定状态。

### 测试范围

#### 核心功能测试
1. **用户认证**
   - 登录/登出流程
   - 密码重置
   - 生物识别登录（Face ID/Touch ID）

2. **仪表板**
   - 数据加载和刷新
   - 统计卡片显示
   - 活动列表

3. **节点管理**
   - 节点列表加载
   - 节点状态显示
   - 节点操作（重启、日志查看）

4. **Playbook管理**
   - Playbook列表
   - 执行Playbook
   - 查看执行日志

5. **设置**
   - 用户信息显示
   - 主题切换
   - 通知设置
   - 安全设置

### 测试要求

1. **参与测试**
   - 接受测试邀请邮件
   - 加入Google Play测试组
   - 从Play Store下载应用

2. **测试反馈**
   - 报告崩溃和错误
   - 提供功能改进建议
   - 记录用户体验问题

3. **测试频率**
   - 每天至少使用一次应用
   - 测试主要功能流程

### 已知限制
- Alpha版本，可能存在不稳定情况
- 部分功能可能需要后端服务支持
- 需要有效的AnixOps账户才能登录

### 反馈方式
- GitHub Issues: https://github.com/anixops/anixops-control-center/issues
- Email: support@anixops.com
